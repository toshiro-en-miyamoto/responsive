Alignment with auto margins

If youâ€™ve ever centered a block by setting the right and left margins to auto, you know that auto margins absorb any available space. By setting both at once, we have two margins that both want all the space, so the block ends up in the middle.

We can apply that to a list of flex items. In the following example, I have a set of flex items displayed per the initial values of flexbox lined up on the left. If I give last-child a left margin of auto, that margin takes up all the space and pushes that one item over to the right. 

            .cards {
                margin: 0 -10px;
                padding: 0;
                list-style: none;
                display: flex;
            }

            .cards li:last-child {
                margin-left: auto;
            }
