Align Grid Items

We can also perform alignment of grid items along the row (or inline) axis with the justify-items property, which sets the justify-self value of the individual grid items.

The initial value of justify-items in Grid Layout is stretch, so the items stretch over their entire area. If we set the value to end, the items shift over to the end of their area — to the right when working in a left-to-right language. 

            .cards {
                …
                justify-items: end;
            }
    
            .one {
                grid-area: a;
            }
    
            .two {
                grid-area: b;
            }
    
            .three {
                grid-area: c;
            }
    
            .four {
                grid-area: d;
                justify-self: stretch;
            }
