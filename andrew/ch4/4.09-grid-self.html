As we did with flexbox, we can use the align-self property to override alignment on individual items.

FIG 4.08: Grid start FIG 4.09: Grid self 
.cards {
    …
    align-items: start;
}

.one {
    grid-area: a;
}

.two {
    grid-area: b;

}

.three {
    grid-area: c;

}

.four {
    grid-area: d;

}
                    
.cards {
    …
    align-items: start;
}

.one {
    grid-area: a;
}

.two {
    grid-area: b;
    align-self: stretch;
}

.three {
    grid-area: c;
    align-self: flex-end;
}

.four {
    grid-area: d;
    align-self: center;
}
