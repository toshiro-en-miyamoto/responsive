I create a four-column and three-row track grid, and lay my items out on the grid using the named areas method I showed you in Chapter 3. 

            .cards {
                margin: 0;
                padding: 0;
                list-style: none;
                display: grid;
                grid-template-columns: repeat(4, 1fr);
                grid-template-rows: repeat(3, 100px);
                grid-template-areas:
                    "a a a b"
                    "a a a b"
                    "c d d d";
                grid-gap: 20px;
                align-items: stretch;
            }
    
            .one {
                grid-area: a;
            }
    
            .two {
                grid-area: b;
            }
    
            .three {
                grid-area: c;
            }
    
            .four {
                grid-area: d;
            }
