Align Flex Items

  • Item 1
  • Item 2
  • Item 3
  • Item 4

If we add the property align-items to the container, we can specify where the items will align with different values:

flex-start
will align items at the top of the container
flex-end
will send them all to the bottom
center
will center them
            .cards {
                margin: 0 -10px;
                padding: 0;
                list-style: none;
                display: flex;
                align-items: flex-start;
                height: 50vh;
            }

The align-items property works on all the items inside the container. But you can override this by using the align-self property on an individual flex item.
