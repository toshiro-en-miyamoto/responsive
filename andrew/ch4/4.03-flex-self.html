Align Flex Items

  • Item 1

  • Item 4

The align-items property works on all the items inside the container. But you can override this by using the align-self property on an individual flex item. 

            .cards {
                margin: 0 -10px;
                padding: 0;
                list-style: none;
                display: flex;
                align-items: flex-start;
                height: 50vh;
            }

            .cards li:nth-child(2) {
                align-self: stretch;
            }
    
            .cards li:nth-child(3) {
                align-self: flex-end;
            }
    
            .cards li:nth-child(4) {
                align-self: center;
            }
