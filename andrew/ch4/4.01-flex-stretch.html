  • Item 1

I have four flex items in a container. The container is set to 50vh, and the flex items stretch to fill it. The items stretch by default because the initial value of align-items is stretch. 

            .cards {
                margin: 0 -10px;
                padding: 0;
                list-style: none;
                display: flex;
                align-items: stretch;
                height: 50vh;
            }
