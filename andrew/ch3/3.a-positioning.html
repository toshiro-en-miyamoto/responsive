position: static
is the initial value for everything that loads on your page; static items are in flow and scroll with the document. We only tend to use this value if we need to reset the position property on an element.
position: relative
With any of the offset properties of top, right, bottom, and left, you can push the element around from its default position. The item that has position: relative on it establishes a new containing block.
position: absolute
An absolutely positioned item is taken out of flow, and can then be positioned from the edge of its containing block
position: fixed
enables items to assume a fixed place on screen as the document loads, and then stay in that place instead of scrolling with the rest of the page. A common reason to use fixed positioning is to keep a menu on screen when scrolling through a long document. Notice that the box is taken out of flow and so will overlap content.
position: sticky
acts as if it is static until the document scrolls to a certain point — at which time it acts as if it is fixed.
