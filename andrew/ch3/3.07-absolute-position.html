If we want this box to be positioned inside the container, and to have the offsets calculated from the edge of the container, we need to establish a containing block on that container. We do this by adding position: relative to .container.

I am absolutely positioned 10 pixels from the top edge of my container and 10 pixels from the right-hand edge.

If I remove height: 250px; in .container, the container will collapse, no longer respecting the height of the element inside. That’s because the box has been taken out of flow, so it doesn’t participate in layout decisions made by those elements. 

        .container {
            width: 400px;
            height: 250px;
            position: relative;
            …
        }
        .box {
            position: absolute;
            top: 10px;
            right: 10px;
            width: 200px;
            …
        }
