One aspect of being inline is that inline elements preserve white space; if they didn’t, every word in a sentence would run together, since the spaces between words would not be maintained. If we use the same method to calculate the size of the item widths as we did for floats, we find that we only get two items on each row. If any white space gets into the document, we’ll end up with only two cards per line again.
